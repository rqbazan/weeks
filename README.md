## Usage

You can also calculate yours based on your birth date, just follow the URL pattern:

```
/:country/:yyyy-mm-dd
```

For example:

- https://weeks.rcrd.space/pe/1996-05-17
- https://weeks.rcrd.space/us/2001-01-01
